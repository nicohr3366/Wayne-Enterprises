# Wayne Industries - Sistema en Java

## Descripción
Sistema de gestión empresarial desarrollado en Java usando Programación Orientada a Objetos.

## Funcionalidades
- Crear departamentos
- Agregar empleados
- Listar departamentos
- Ver empleados por departamento
- Eliminar empleados
- Mostrar toda la empresa

## Cómo ejecutar

1. Compilar:
javac *.java

2. Ejecutar:
java Main
